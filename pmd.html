Work Overview:

Piano Music Database (PMD) is a free online tool that helps you easily & quickly find music for your piano students. Search for pieces that incorporate specific pedagogical concepts; filter your results by mood, theme, style, and more!

Initial versions of the website were made using WordPress to test the feasibility of a piano music database website on mobile and desktop. Currently the website is a "Next.js with Tailwind CSS" PWA/website which is deployed from a GitHub repository using Netlify. The search engine is powered by Typesense and the work data is gathered from a Strapi API.

The current branding (logo, font, color, social media post/email newsletter images, banners, etc) is designed using Adobe Illustrator and Adobe Photoshop.
